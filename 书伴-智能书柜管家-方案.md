# 书伴 · 智能书柜管家

## 概述

一款面向纸质书读者的Android应用，通过扫描图书ISBN条形码一键录入藏书，利用Google Books API自动获取书籍信息，并结合LLM大模型生成个性化书单推荐，帮助用户轻松管理个人书柜、发现下一本好书。

---

## 核心功能点

| 功能模块 | 说明 |
|---------|------|
| **ISBN扫码录入** | 调用ZXing库扫描图书背面条形码，解析ISBN |
| **书籍信息自动获取** | 通过ISBN调用Google Books API获取书名、作者、封面、出版社、简介等信息 |
| **个人书柜管理** | 书架展示、分类筛选（想读/在读/已读）、删除藏书 |
| **阅读状态追踪** | 标记每本书的阅读状态，记录阅读进度 |
| **AI书单推荐** | 将用户藏书列表发送给LLM，获取个性化推荐；支持输入心情/偏好进行对话式推荐 |
| **读书笔记** | 每本书可添加私人笔记，支持编辑和查看 |
| **主题切换** | 日间/夜间模式 |
| **API Key配置** | 用户在设置页填入自己的LLM API Key（OpenAI兼容格式），本地安全存储 |

## 页面设计

### 1. 启动页 (`SplashActivity`)
- Logo + 应用名称「书伴」
- 自动跳转主页

### 2. 主页 (`MainActivity`)
- **顶部**：欢迎语 + 统计卡片（藏书总数、在读数量、已读数量）
- **中部**：最近添加的书籍（横向RecyclerView，展示封面）
- **底部推荐卡**：「AI今日推荐」入口按钮 → 跳转推荐页
- **底部导航栏**：首页 / 书柜 / 推荐 / 我的

### 3. 扫码页 (`ScanActivity`)
- **中央取景框**：扫描区域指示
- **底部文字**：「将条形码置于框内自动识别」
- 扫描成功 → 弹出加载弹窗 → 跳转书籍详情页
- 扫描失败 → Toast提示 + 支持手动输入ISBN

### 4. 书籍详情页 (`BookDetailActivity`)
- **顶部**：书籍封面大图
- **信息区**：书名、作者、出版社、出版日期、页数、豆瓣评分(如有)
- **简介区**：书籍内容简介（Google Books返回）
- **操作区**：
  - 「加入书柜」按钮（已加入则显示「已在书柜」）
  - 阅读状态选择器：想读 / 在读 / 已读
  - 「写笔记」入口 → 跳转笔记编辑页
- **底部**：「AI聊聊这本书」按钮 → 调用LLM，返回关于此书的讨论/书评

### 5. 书柜页 (`BookshelfActivity`)
- **顶部Tab筛选**：全部 / 想读 / 在读 / 已读
- **主体**：书籍网格列表（瀑布流，显示封面+书名）
- 点击某本书 → 跳转详情页
- 长按 → 弹出删除确认对话框
- **搜索按钮**：在书柜内按书名搜索

### 6. 推荐页 (`RecommendActivity`)
- **输入区**：用户输入当前心情/偏好的关键词（如"最近喜欢悬疑类"）
- **推荐结果列表**：LLM返回的书名+作者+推荐理由
- 每条推荐结果右侧有「查看详情」按钮 → 通过Google Books API搜索该书 → 跳转详情页
- **顶部快捷入口**：「根据我的书柜推荐」（不填输入，直接发送收藏列表给LLM）

### 7. 笔记编辑页 (`NoteActivity`)
- **标题栏**：「《书名》· 笔记」
- **编辑区**：多行文本输入框
- **保存按钮**：存入Room数据库，时间戳记录

### 8. 设置页 (`SettingsActivity`)
- **LLM API配置**：
  - API Endpoint输入框（默认OpenAI格式）
  - API Key输入框（密文显示）
  - 模型名称输入（如gpt-4o）
- **主题切换**：浅色 / 深色 / 跟随系统
- **关于**：版本号、使用说明

## 技术实现要点

| 技术点 | 对应功能 |
|-------|---------|
| **Camera + ZXing** | ISBN条形码扫描 |
| **OkHttp / Retrofit** | Google Books API、LLM API的网络请求 |
| **Gson / Moshi** | JSON数据解析（API返回） |
| **Room数据库** | 藏书信息、阅读状态、笔记的本地持久化 |
| **SharedPreferences** | API Key存储、主题设置、用户偏好 |
| **RecyclerView (多视图类型)** | 书籍列表、推荐列表 |
| **Glide / Picasso** | 书籍封面图片加载 |
| **ViewPager + TabLayout** | 书柜页的状态筛选Tab |
| **AsyncTask / ExecutorService** | 异步网络请求、数据库操作 |
| **BroadcastReceiver** | 网络状态监听 |
| **Material Design** | 主题样式、日夜间模式切换 |
| **Activity生命周期** | 各页面状态保存与恢复 |

## API调用说明

### Google Books API
- 无需API Key即可基础调用（有限额）
- 接口：`GET https://www.googleapis.com/books/v1/volumes?q=isbn:{ISBN}`
- 返回：书名、作者、封面图URL、出版社、出版日期、页数、简介

### LLM API（OpenAI兼容格式）
- 用户自行填入Endpoint + Key + Model
- 请求格式：
```json
{
  "model": "用户配置的模型名",
  "messages": [
    {"role": "system", "content": "你是一位专业的图书推荐助手..."},
    {"role": "user", "content": "用户输入/藏书列表"}
  ],
  "temperature": 0.7
}
```
- 用于：AI书单推荐、单书讨论

```
Book（书籍）：
  id(自增) | isbn | title | author | coverUrl | publisher | 
  publishDate | pageCount | description | status(想读/在读/已读) | 
  addedTime

Note（笔记）：
  id(自增) | bookId(外键) | content | updateTime
```
